# agenticenghack

A hackathon for agentic engineering — building the tools, agents, and evals that make AI a real part of the engineering loop.

## Theme

We're past the demo phase. The interesting question now is: **what does engineering actually look like when agents are first-class collaborators?**

Bring whatever you want to build. A few directions worth exploring:

- **Agent tooling** — better hooks, skills, MCP servers, slash commands
- **Evals & benchmarks** — how do we measure agent quality on real engineering work
- **Multi-agent workflows** — orchestration, hand-offs, parallelism, verification
- **Developer UX** — IDE integrations, terminal experiences, review workflows
- **Infrastructure** — sandboxes, durable execution, observability for agents
- **Wildcards** — anything that surprises us

## How to participate

1. Read [CONTRIBUTING.md](./CONTRIBUTING.md) for how to add your project
2. Drop your project under `projects/<your-project-name>/`
3. Include a short README explaining what it does and how to run it
